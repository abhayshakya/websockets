hello, this is a user session chat app, one time chat-app,

step 1: register

step 2: login

step 3: join room, write room name yourself and let other user join that room name.

step 4: chat until u logout, once logout, repeat step 1.



---------------------- FOR SETUP -------------------

use git clone https://github.com/abhayshakya/websockets.git

npm install

npm run dev

-------------------- FOR CHAT IN TERMINAL --

for terminal chat npm run dev then open another terminal

do npm run dev2 or node client.js

-> enter username

-> enter password

-> will show connected then

-> use syntax: room your-roomnam -> enter

-> for chat use: chat your chat-message
